Table panel

The table here displays predicted probabilites of the given entries (both Wild type sequences and mutations) by PRECOGx and known values from IUPHAR and the Shedding/ebBRET assays.
For rows displaying:
VAR = IUPHAR: primary coupling is represented by PC while the secondary ones as SC and the unknown ones as -.
VAR = LogRAi (Shedding experiment): values > -1.0 are coupling while others are not coupling (< -1.0) or unknown -.
VAR = Emax (ebBRET experiment): values > 0 are coupling while others are not coupling (= 0) or unknown -.
VAR = WT(Wild Type)/Mutation: values with predicted probability (by PRECOGx) greater than 0.5 are coupling while others are not coupling.

By clicking on any of the cells:
a) The corresponding input is shown on the PCA panel
b) The enriched/depleted contact positions of the input GPCR are highlighted in the Conact/Sequence panels and mapped onto a 3D structure in the Structure panel.

PCA panel

This panel displays the first two eigenvectors (X & Y-axis) of the Principal Component Analysis (PCA) applied on the embeddings of the human GPCRome, comprising a total of 377 receptors (279 Class A, 33 Class B, 17 class C, 11 class F, 25 Taste receptors and 14 in other classes). Every GPCR and the input (WT/mutation) can be annotated with coupling information from either Shedding, ebBRET, IUPHAR or STRING (only for B-arrs).

Contact/Sequence panel

This panel displays the computed differential contact maps (Contacts panel) for the selected G-protein group (from the table above), and the same conacts being mapped to the input GPCR sequence (Sequence Panel). These conacts are either enriched (log-odds score > 0) or depeleted (log-odds score < 0).
Hover on the sequence to to view the corresponding position's Ballesteros Weinstein (BW) annotation.
Click on the button to toggle between the panels.
Use the slider the set the absolute value cutoff of the Log-odds score. For example, a slider value of 0.3 will display the contacts that have an enriched (> 0.3) or depeleted (< -0.3) for the given G-potein coupling score.

Structure panel

This panel displays the differential contacts mapped to the input GPCR's 3D structure. Zoom-in to view the Ballesteros Weinstein (BW) annotation of the positions.
